DESCRIPTION - This code is used to predict wins and rank of a season. 
INSTALLATION - 
    1. Install all required packages to run the code (in the requirements.txt file in the code folder)
    2. Collect desired data. Using the Data Collection and cleaning, select the year you would like to predict by changing the desired_year variable to the year you want.
    3. To see Vizualizations, open Interactive Visualizations.twb in Tableau. You will need to set the datasource as the visuals_df.csv located in the data folder.
EXECUTION - How to run a demo on your code
    Open 2020 Predictions.ipynb file and run all cells. That will run the data Collection and cleaning, and train the models for you. and output the predicted Wins and Ranks based on your desired year using our Models
